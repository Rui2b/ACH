# 🤖 ACH (AI Command Helper) - 让你的终端听懂人话

**ACH** 是一款专为高效（和懒惰）设计的本地 AI 命令行工具。它将一个 **5亿参数的 AI 大脑** 直接封装进程序中。

无需安装 Python 环境，无需配置复杂的 Ollama，无需连接互联网。只需一个二进制文件，即可通过“自然语言”指挥你的 Windows、Linux 或 macOS 执行系统任务。

---

## 🌟 核心亮点：为什么选择 ACH？

* **📦 零依赖，双击即用**：不需要安装任何运行库或环境，真正的“解压即用”。
* **🔒 隐私至上**：100% 本地推理。你的指令、路径和私密文件名永远不会离开你的电脑。
* **🧠 内置大脑**：集成 `Qwen2.5-0.5B` 轻量化模型，针对系统指令进行了专门调优。
* **⚡ 极致轻量**：运行时仅需约 500MB 内存，即便是老旧电脑也能流畅指挥。
* **🌍 全平台支持**：一份代码，三端通用。

---

## ⚠️ 避坑指南与注意事项
加载等待：由于 AI 模型需要从硬盘加载到内存，程序启动后的前 3-5 秒可能没有反应。这是正常现象，请稍等片刻。

文件体积：因为我们打包了整个 AI 模型文件，程序体积在 400MB - 500MB 左右。这虽然大，但换来了“离线可用”的极致便利。

二次确认：AI 不是万能的。在按下 y 执行前，请务必检查 AI 建议的命令是否安全，避免误删重要资料。

模型限制：内置的 0.5B 模型擅长文件处理、目录操作、系统状态查询等。如果要求它写复杂的 Python 脚本，它可能会因为“脑容量”不足而报错。
